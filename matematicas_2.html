    return resultado

Límites

sympy

Límite en un punto

Valor al que se aproxima f(x) cuando x tiende a un punto.

sp.limit(f, x, 0)        # x → 0
sp.limit(f, x, sp.oo)   # x → ∞
sp.limit(f, x, -sp.oo)  # x → -∞

Si el límite existe, los laterales deben coincidir.

sympy

Límites laterales

A veces la función se aproxima a valores distintos por cada lado.

sp.limit(f, x, 3, '+')  # por la derecha
sp.limit(f, x, 3, '-')  # por la izquierda

Si lim(+) ≠ lim(−) → el límite no existe en ese punto.

sympy

Límites iterados (2 variables)

Para f(x, y) en el punto (a, b): primero un límite, luego el otro.

f = x**2 / (x + y)

lim_xy = sp.limit(sp.limit(f, y, 2), x, 1)
lim_yx = sp.limit(sp.limit(f, x, 1), y, 2)

El orden importa. Si lim_xy ≠ lim_yx, el límite doble puede no existir.

sympy

Definición de derivada como límite

La derivada es un límite cuando h → 0.

from sympy.abc import x, h

f = sp.cos(x)
sp.limit((f.subs(x, x+h) - f) / h, h, 0)

Esto equivale a sp.diff(f, x).

Derivadas parciales

Qué es una derivada parcial

Para funciones de varias variables, la derivada parcial mide cómo cambia f cuando varía solo una variable, manteniendo las demás constantes.

Análisis de funciones

Pasos para analizar una función

1
Asíntotas horizontales

Límite cuando x → ±∞. Si da un número finito, hay asíntota horizontal en ese valor.

2
Asíntotas verticales

Igualar el denominador a cero. Las raíces son candidatas a asíntotas verticales.

3
Puntos críticos

Calcular f'(x) e igualar a cero. Las soluciones son candidatos a máximos o mínimos.

4
Clasificar máximos/mínimos

Evaluar f''(x) en cada punto crítico. Si f'' > 0 → mínimo. Si f'' < 0 → máximo.

Pasos para resolver un problema de optimización

1
Identificar las variables

¿Qué dimensiones o cantidades intervienen?

2

TEMA 03

Sumas de Riemann

Aproximan el área bajo una curva dividiendo el intervalo en n rectángulos. Cuantos más intervalos, más preciso.

Ln — Suma izquierda

Altura = extremo izquierdo de cada intervalo

import numpy as np
import sympy as sp

x = sp.symbols('x')

def L_n(f, a, b, n):
    xvalues = np.linspace(a, b, n)
    yvalues = sp.lambdify(x, f)(xvalues[:-1])  # quita el último punto
    Lsum = 0
    for i in yvalues:
        Lsum += i
    Ax = (b - a) / n
    return Ax * Lsum

xvalues[:-1] — usa todos los puntos menos el último (extremos izquierdos).

Rn — Suma derecha

Altura = extremo derecho de cada intervalo

def R_n(f, a, b, n):
    xvalues = np.linspace(a, b, n)
    yvalues = sp.lambdify(x, f)(xvalues[1:])   # quita el primer punto
    Rsum = 0
    for i in yvalues:
        Rsum += i
    Ax = (b - a) / n
    return Ax * Rsum

xvalues[1:] — usa todos los puntos menos el primero (extremos derechos).

Diferencia clave entre Ln y Rn

MétodoÍndices de xvaluesPuntos usados
Suma izquierda Lnxvalues[:-1]x₁, x₂, ..., xₙ (sin el último)
Suma derecha Rnxvalues[1:]x₂, x₃, ..., xₙ₊₁ (sin el primero)

Trapecio y Simpson

Suma trapezoidal — manual

Promedia los dos extremos de cada intervalo

def suma_trapezoidal(f, a, b, n):
    Ax = (b - a) / n
    x_values = np.linspace(a, b, n)
    y_values = sp.lambdify(x, f)(x_values)

    total = 0
    for i in range(len(y_values) - 1):
        total += (y_values[i] + y_values[i+1]) / 2

    return Ax * total

Más precisa que Ln y Rn para el mismo n porque usa ambos extremos de cada subintervalo.

Trapecio y Simpson con scipy

Versión directa con numpy/scipy

from scipy import integrate
import numpy as np

def f(x):
    return np.sin(x)

a, b, n = 0, np.pi, 10
x = np.linspace(a, b, n+1)
y = f(x)

# Trapecio
trapecio = np.trapz(y, x)

# Simpson (n debe ser par)
simpson = integrate.simps(y, x)

Simpson usa polinomios cuadráticos en cada subintervalo — más preciso que el trapecio. n debe ser par.

Integrales numéricas con scipy

from scipy.integrate import quad

Fórmula

Mide cuánto se aleja la aproximación del valor exacto, en proporción al valor exacto:

simpson  = integrate.simps(y, x)

# Errores relativos
error_trapecio = abs(valor_exacto - trapecio) / abs(valor_exacto)
error_simpson  = abs(valor_exacto - simpson)  / abs(valor_exacto)

print(f"Exacto:   {valor_exacto:.10f}")
print(f"Trapecio: {trapecio:.10f} | Error: {error_trapecio:.2e}")
print(f"Simpson:  {simpson:.10f}  | Error: {error_simpson:.2e}")

Gráficas simbólicas — symplot

Para funciones simbólicas de sympy. No necesitas np.linspace, sympy discretiza sola.

básico

Gráfica simbólica simple

import sympy as sp
from sympy.plotting import plot as symplot

x = sp.symbols('x')

symplot(x**2, (x, -5, 5))

(x, -5, 5) es el rango de representación: variable, mínimo, máximo.

dos funciones

Función y su derivada

x = sp.symbols('x')
f = x**2
df = sp.diff(f, x)   # → 2x

p = symplot(f, df, (x, -5, 5), show=False)
p[0].line_color = 'red'   # color de f
p.show()

show=False evita que se dibuje antes de tiempo. Así puedes modificar colores antes de p.show().

con límites de ejes

Limitar los ejes

p = symplot(f, xlim=(-6, 6), ylim=(-12, 12))
p.show()

Útil cuando la función tiene asíntotas y se dispara hacia infinito.

asíntota

Función con asíntota horizontal

x = sp.symbols('x')
f = 2*(x**2 - 9) / (x**2 - 4)

# Dibuja f y la línea y=2 (asíntota horizontal)
p = symplot(f, 2, xlim=(-6, 6), ylim=(-12, 12), show=False)
p.show()

Puedes pasar un número (como 2) y symplot lo dibuja como línea horizontal.

¿Cuándo uso cada una?

Regla general

Depende de cómo tengas definida la función:

Si la función es... Usa Por qué
Numérica con def o lambda plt.plot Ya devuelve números, solo necesitas linspace
Simbólica con sympy symplot o lambdify + plt.plot symplot es más directo; lambdify si quieres marcar puntos
Simbólica y quieres marcar puntos críticos lambdify + plt.plot symplot no permite añadir puntos fácilmente

Aviso importante: x vs x1

Si tienes x definida como símbolo de sympy, no puedes usarla también como array de numpy en la misma celda. Por eso en el notebook verás x1 e y1 para las gráficas numéricas.
