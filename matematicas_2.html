TEMA 02

Límites

sympy

Límite en un punto

Valor al que se aproxima f(x) cuando x tiende a un punto.

sp.limit(f, x, 0)        # x → 0
sp.limit(f, x, sp.oo)   # x → ∞
sp.limit(f, x, -sp.oo)  # x → -∞

Si el límite existe, los laterales deben coincidir.

sympy

Límites laterales

A veces la función se aproxima a valores distintos por cada lado.

sp.limit(f, x, 3, '+')  # por la derecha
sp.limit(f, x, 3, '-')  # por la izquierda

Si lim(+) ≠ lim(−) → el límite no existe en ese punto.

sympy

Límites iterados (2 variables)

Para f(x, y) en el punto (a, b): primero un límite, luego el otro.

f = x**2 / (x + y)

lim_xy = sp.limit(sp.limit(f, y, 2), x, 1)
lim_yx = sp.limit(sp.limit(f, x, 1), y, 2)

El orden importa. Si lim_xy ≠ lim_yx, el límite doble puede no existir.

sympy

Definición de derivada como límite

La derivada es un límite cuando h → 0.

from sympy.abc import x, h

f = sp.cos(x)
sp.limit((f.subs(x, x+h) - f) / h, h, 0)

Esto equivale a sp.diff(f, x).

Ejemplos típicos del notebook

x = sp.symbols('x')

sp.limit(sp.sin(x)/x, x, 0)              # → 1
sp.limit(1/x, x, 0, '+')                 # → ∞
sp.limit(1/x, x, 0, '-')                 # → -∞

f = (x - 3) / sp.Abs(x - 3)
sp.limit(f, x, 3, '+')                   # → 1
sp.limit(f, x, 3, '-')                   # → -1

Derivada

sympy

Calcular la derivada

x = sp.symbols('x')
f = x**2

sp.diff(f, x)       # → 2x (primera)
sp.diff(f, x, 2)    # → 2  (segunda)
sp.diff(f, x, 3)    # → 0  (tercera)
sympy

Expresión matemática de la derivada

d = sp.Derivative(x**2, x)
d         # muestra d/dx(x²)
d.doit()  # calcula → 2x

Útil para mostrar la notación matemática sin calcularla todavía.

sympy

Evaluar la derivada en un punto

df = sp.diff(x**2, x)   # → 2x

df.subs(x, 3)           # → 6
df.evalf(subs={x: 3})   # → 6.0

Derivadas parciales

sympy

Qué es una derivada parcial

Para funciones de varias variables, la derivada parcial mide cómo cambia f cuando varía solo una variable, manteniendo las demás constantes.

x, y = sp.symbols('x y')
g = x**2 + y**3

sp.diff(g, x)    # → 2x  (respecto a x)
sp.diff(g, y)    # → 3y² (respecto a y)
sympy

Segundas derivadas parciales

sp.diff(g, x, 2)   # → 2   (∂²g/∂x²)
sp.diff(g, y, 2)   # → 6y  (∂²g/∂y²)
sp.diff(g, x, y)   # → 0   (mixta ∂²g/∂x∂y)
sympy

Gradiente

Vector formado por todas las derivadas parciales de una función escalar.

f = x**2 * y**3

grad_x = sp.diff(f, x)   # → 2xy³
grad_y = sp.diff(f, y)   # → 3x²y²

gradiente = [grad_x, grad_y]
sympy

Evaluar en un punto

f = 8*x**2 + 2*y**2 - 4*x*y + 3*y - 4*x + 5

df_x = sp.diff(f, x)
df_y = sp.diff(f, y)

df_x.subs([(x, 1), (y, -2)])
df_y.subs([(x, 1), (y, -2)])

Análisis de funciones

Pasos para analizar una función

1
Asíntotas horizontales

Límite cuando x → ±∞. Si da un número finito, hay asíntota horizontal en ese valor.

2
Asíntotas verticales

Igualar el denominador a cero. Las raíces son candidatas a asíntotas verticales.

3
Puntos críticos

Calcular f'(x) e igualar a cero. Las soluciones son candidatos a máximos o mínimos.

4
Clasificar máximos/mínimos

Evaluar f''(x) en cada punto crítico. Si f'' > 0 → mínimo. Si f'' < 0 → máximo.

asíntotas

Asíntotas en código

f = 2*(x**2 - 9) / (x**2 - 4)

sp.limit(f, x, sp.oo)    # asíntota horizontal

den = x**2 - 4
sp.solve(den, x)          # asíntotas verticales: x=±2
puntos críticos

Puntos críticos en código

f1 = sp.diff(f, x)
criticos = list(sp.solveset(f1, x))

f2 = sp.diff(f, x, 2)
for cr in criticos:
    val = f2.subs(x, cr)
    # val > 0 → mínimo
    # val < 0 → máximo

solveset

solveset vs solve

El notebook usa sp.solveset para encontrar raíces porque maneja mejor los dominios.

sp.solveset(f1, x)
list(sp.solveset(f1, x, domain=sp.S.Reals))

Añadir domain=sp.S.Reals descarta soluciones complejas.

Optimización

Pasos para resolver un problema de optimización

1
Identificar las variables

¿Qué dimensiones o cantidades intervienen?

2
Encontrar la función a optimizar

Coste, área, volumen... expresarla en función de una sola variable usando las restricciones del problema.

3
Calcular los puntos críticos

Derivar e igualar a cero. Resolver para x.

4
Verificar que tiene sentido físico

Descartar soluciones negativas o fuera del dominio del problema.

5
Confirmar máximo o mínimo

Usar la segunda derivada o evaluar en los extremos del dominio.

simbólico

Optimización con sympy

x = sp.symbols('x')
coste = 60*x**2 + 800/x

derivada = sp.diff(coste, x)
criticos = list(sp.N(sp.solveset(
    derivada, x, domain=sp.S.Reals)))

criticos = [c for c in criticos if c >= 0]
scipy

Optimización con scipy

from scipy import optimize

def f(x):
    return 60*x**2 + 800/x

optimize.fmin(f, 1)           # desde x=1
optimize.fminbound(f, -1, 2)  # en [-1, 2]

scipy y sympy son incompatibles. Con scipy hay que usar funciones def estándar, no expresiones simbólicas.
