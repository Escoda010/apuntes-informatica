LECCIÓN 01

Chuleta completa matemática discreta — Lección 1

Decimal → base t
1
Divide el número entre t
Anota el cociente y el resto
2
Divide el cociente entre t
Repite hasta que el cociente sea 0
3
Lee los restos de abajo hacia arriba
Ese es el número en base t
Ojo: son los RESTOS, no los cocientes
Ej: 137 en base 2
137 = 2·68 + 1 → r=1
68 = 2·34 + 0 → r=0
34 = 2·17 + 0 → r=0
17 = 2·8 + 1 → r=1
8 = 2·4 + 0 → r=0
4 = 2·2 + 0 → r=0
2 = 2·1 + 0 → r=0
1 = 2·0 + 1 → r=1

Resultado: (10001001)₂
Base t → decimal
1
Multiplica cada dígito por t elevado a su posición
El dígito más a la derecha es la posición 0
2
Suma todo
Ej: (4165)₇
4·7³ + 1·7² + 6·7¹ + 5·7⁰
= 1372 + 49 + 42 + 5 = 1468
Algoritmo de Euclides — mcd(a, b)
1
Divide a entre b
a = b·q + r   con 0 ≤ r < b
2
Ahora divide b entre r
El divisor pasa a ser el resto anterior
3
Repite hasta que el resto sea 0
4
mcd = último resto distinto de 0
El cociente siempre es entero — quédate con la parte entera
Ej: mcd(56, 34)
56 = 34·1 + 22
34 = 22·1 + 12
22 = 12·1 + 10
12 = 10·1 + 2
10 = 2·5 + 0 ← resto 0

mcd(56, 34) = 2
Bézout — encontrar s, t tal que mcd = a·s + b·t
1
Localiza el mcd en la tabla de Euclides
Está en el penúltimo paso (donde aparece como resto)
2
Despeja el mcd
mcd = número grande − número pequeño · cociente
3
Sube un paso, sustituye el resto más pequeño
Búscalo en el paso anterior de Euclides y sustitúyelo
4
Agrupa coeficientes y repite
Hasta que solo queden a y b
Siempre eliminas el número más pequeño — vas de abajo hacia arriba
Ej: mcd(56,34) = 2, s y t = ?

2 = 12 − 10·1
= 12 − (22 − 12·1) = 2·12 − 22
= 2·(34−22·1) − 22 = 2·34 − 3·22
= 2·34 − 3·(56−34·1)
= 56·(−3) + 34·5

s = −3, t = 5
Verificación: 56·(−3)+34·5 = −168+170 = 2 ✓
Ecuación diofántica — ax + by = c
1
¿Tiene solución?
Calcula mcd(a,b). Si mcd divide a c → hay solución. Si no → no hay solución.
2
Calcula s, t con Bézout
mcd = a·s + b·t
3
Solución particular
k = c / mcd  →  x₀ = k·s, y₀ = k·t
4
Calcula α y β
α = a / mcd    β = b / mcd
5
Solución general
x = x₀ + β·k    y = y₀ − α·k    k ∈ ℤ
Ojo con los signos en la solución general: x suma β, y resta α
Ej: 56x + 20y = 8

mcd(56,20) = 4, 4|8 → hay solución
Bézout: 56·(−1) + 20·3 = 4
k = 8/4 = 2 → x₀=−2, y₀=6
α=14, β=5

x = −2 + 5k
y = 6 − 14k k ∈ ℤ
MCD y MCM por factorización prima
1
Descompón cada número en factores primos
Divide entre 2, 3, 5, 7... hasta llegar a 1
2
MCD → mínimo exponente de cada primo común
3
MCM → máximo exponente de todos los primos
Ej: a=180, b=126
180 = 2²·3²·5
126 = 2¹·3²·7

mcd = 2¹·3² = 18 (mínimos)
mcm = 2²·3²·5·7 = 1260 (máximos)

Verificación: 180·126 = 18·1260 = 22680 ✓
Fórmula clave

a · b = mcd(a,b) · mcm(a,b)

Si conoces tres de los cuatro valores, puedes despejar el cuarto.

mcm = (a · b) / mcd
mcd = (a · b) / mcm
a = (mcd · mcm) / b
